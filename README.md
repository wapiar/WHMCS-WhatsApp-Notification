## Installation

### Watch Guide Video  
Before installing, it’s highly recommended to watch the integration tutorial above.

### Manual Installation  

1. Upload the module files to your **WHMCS root directory**  
2. Go to **WHMCS Admin → Setup → Addon Modules → Activate WAPIAR Notification**  
3. Enter your **WAPIAR API Key** (get from [WAPIAR Dashboard](https://wapiar.com))  
4. Configure desired notification events and templates  
5. Test gateway connection — you should see `{Success:true}`  

> Note: Current module doesn’t support emojis.  
> Version 2.0 is in development with **emoji and media support**.
